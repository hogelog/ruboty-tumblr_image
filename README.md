# Ruboty::TumblrImage
An ruboty handler to pickup images from Tumblr.

## Usage
```
@ruboty tumblr <username> - Pickup image from Tumblr
@ruboty tumblr unsafe <username> - Pickup image from Tumblr (unsafe)
```
